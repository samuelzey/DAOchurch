## DAOchurch社区是什么
### 社区
DAOchurch首先是一个社区，希望用去中心化的力量赋能现有组织模式，推动DAO在世界范围内的使用。
### 工具
作为一个去中心化的社区，我们必须有完全满足我们自己需求的社区工具。
### 不断自增长的实体
DAO作为一个新兴领域，有足够多的需求等待被满足，DAOchurch仍然在不断成长，希望在这个领域做出自己的独特贡献。
